🚀 Excited to share my latest project — Gayanganga Library Management System!

I built a complete, production-ready Library Management Web Application from scratch using ASP.NET Core MVC (.NET 8), deployed live on the cloud. Here's what's packed inside:

📚 Key Features:
✅ Student Management — Add, Edit, Delete students with photo & Aadhaar card upload
✅ Attendance System — Dashboard with 75-seat grid (green/red tiles), daily attendance marking, student self-attendance via public link
✅ Fees Management — Monthly payment tracking with WhatsApp reminder for pending fees
✅ Water & Paper Tracker — Daily consumption tracking with monthly printable reports
✅ Reports Module — Student, Attendance & Fees reports with print functionality
✅ Settings — Theme customization, password change, library info management
✅ PWA Support — Works offline, installable on mobile & desktop
✅ Secure Login — Cookie-based authentication, session management

🛠️ Tech Stack:
• Backend: ASP.NET Core MVC (.NET 8)
• Database: MS SQL Server (EF Core, Code First)
• Frontend: Bootstrap 5, Font Awesome, Vanilla JS (AJAX)
• Hosting: Somee.com (Live!)
• Features: PWA, Responsive Design, Print Reports, WhatsApp Integration

🌐 Live Demo: http://gayangangalibrary.somee.com

This project taught me end-to-end development — from database design and backend APIs to responsive UI, AJAX interactions, authentication, cloud deployment, and PWA configuration.

Looking forward to building more impactful solutions! 💪
